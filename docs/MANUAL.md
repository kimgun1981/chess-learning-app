## 게임 시작

### 1단계 — 모드 선택

| 버튼 | 설명 |
|------|------|
| 🤖 1인용 (AI 연습) | AI와 1:1 대결. 힌트·해설 지원 |
| 👥 2인용 | 같은 기기로 두 명이 대결 |

---

### 2단계 — 세부 설정

**1인용:** 난이도를 선택합니다

| 난이도 | 특징 |
|--------|------|
| 🌱 초급 | AI가 무작위로 이동. 규칙 익히기에 적합 |
| ⚡ 중급 | AI가 기물 잡기·체크·중앙 장악을 우선 |
| 🔥 심화 | AI가 3수 앞까지 계산하는 미니맥스 알고리즘 사용 |

**2인용:** 제한 시간을 선택합니다

| 시간 | 유형 |
|------|------|
| ∞ 무제한 | 시간 제한 없음 |
| ⚡ 1분 | 불릿 (초고속) |
| 🔥 3분 | 블리츠 (빠름) |
| ⏱ 5분 | 블리츠 (표준) |
| 🕐 10분 | 래피드 |
| ♟ 30분 | 클래식 |

## 게임 조작 방법

### 기물 이동

1. **이동할 기물을 클릭** → 기물이 하이라이트되고 이동 가능한 칸에 파란 점 표시
2. **이동할 목적지 칸을 클릭** → 이동 완료

> 내 차례가 아닌 기물은 선택되지 않습니다.

### 선택 취소

- 다른 빈 칸 또는 이동 불가 칸을 클릭하면 선택이 해제됩니다.

## 화면 구성 (1인용)

```
┌─────────────────────┬──────────────────┐
│                     │  현재 상태 / 팁   │
│                     ├──────────────────┤
│                     │  🤖 AI 해설       │
│      체 스 판        ├──────────────────┤
│                     │  기물 설명        │
│                     ├──────────────────┤
│                     │  📋 기보          │
│                     ├──────────────────┤
│                     │ 💡힌트 ↩무르기 🔄 │
└─────────────────────┴──────────────────┘
```

### 사이드 패널 기능

| 항목 | 설명 |
|------|------|
| 현재 상태 / 팁 | 체크 경고 또는 단계별 전략 팁 |
| AI 해설 | AI가 방금 둔 수의 의도 설명 |
| 기물 설명 | 클릭한 기물의 이동 규칙 |
| 기보 | 백/흑 착수 기록 (SAN 표기) |
| 💡 힌트 | 최선 수를 노란색으로 표시 (백 차례에만 활성) |
| ↩ 무르기 | 내 수 + AI 수 한 쌍 취소 |
| 🔄 초기화 | 게임을 처음부터 다시 시작 |

## 화면 구성 (2인용)

```
┌─────────────────────────────────┐
│   흑 (Black) 카드 [뒤집혀 표시]  │  ← 흑 플레이어가 읽는 방향
├─────────────────────────────────┤
│                                 │
│           체 스 판               │
│                                 │
├─────────────────────────────────┤
│         백 (White) 카드          │
├─────────────────────────────────┤
│       [↩ 무르기]  [🔄 새 게임]   │
└─────────────────────────────────┘
```

### 플레이어 카드

- 자기 차례가 되면 카드 테두리가 초록색으로 강조되고 **● 차례** 표시
- 타이머가 설정된 경우 남은 시간 실시간 표시
- 30초 이하이고 자기 차례이면 빨간색으로 깜빡임
- 잡은 기물 목록 및 점수 우위 표시

### 2인용 배치 권장

iPad를 테이블 위에 평평하게 놓고 양쪽에 앉습니다.
- 흑 플레이어: 화면 위쪽에 앉음 (흑 카드가 자신 방향으로 읽힘)
- 백 플레이어: 화면 아래쪽에 앉음 (백 카드가 자신 방향으로 읽힘)
- 보드는 rank 8이 위, rank 1이 아래로 고정 (차례가 바뀌어도 변하지 않음)

## 체스 기물 이동 규칙

| 기물 | 흰 말 | 검정 말 | 이동 방법 |
|------|-------|---------|---------|
| 킹 | ♔ | ♚ | 모든 방향 1칸 |
| 퀸 | ♕ | ♛ | 직선·대각선 모든 방향 |
| 룩 | ♖ | ♜ | 직선(상하좌우) |
| 비숍 | ♗ | ♝ | 대각선 |
| 나이트 | ♘ | ♞ | L자형 (2+1), 뛰어넘기 가능 |
| 폰 | ♙ | ♟ | 앞으로 1칸 (첫 수 2칸 가능), 공격은 대각선 |

### 특수 이동

| 이름 | 조건 | 방법 |
|------|------|------|
| 캐슬링 | 킹·룩 미이동, 사이 칸 비어있음, 체크 없음 | 킹을 룩 방향 2칸 이동 |
| 앙파상 | 상대 폰이 2칸 전진한 직후 | 측면 폰으로 대각선 잡기 |
| 프로모션 | 폰이 상대 끝줄 도달 | 자동으로 퀸으로 승격 |

## 개발 / 운영 가이드

### 로컬 실행

```bash
# 저장소 클론
git clone https://github.com/kimgun1981/chess-learning-app.git
cd chess-learning-app

# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:5173)
npm run dev
```

### 빌드

```bash
npm run build
# dist/ 폴더에 빌드 결과물 생성
```

### 배포 (GitHub Pages)

```bash
npm run deploy
# 자동으로: vite build → git commit → git push → gh-pages 브랜치 배포
```

### 주요 수정 포인트

| 수정 목적 | 파일 | 위치 |

## 문제 해결

| 증상 | 해결 방법 |
|------|---------|
| 화면이 이전 버전으로 보임 | Cmd+Shift+R (Mac) / Ctrl+Shift+R (Win)으로 강력 새로고침 |
| AI가 응답하지 않음 | 심화 난이도는 계산에 1~2초 소요됨. 정상 동작입니다 |
| 기물이 이동되지 않음 | 내 차례인지 확인. 상대 기물은 선택 불가 |
| 시간이 줄어들지 않음 | 무제한 모드로 설정된 상태입니다 |
