# 체스 교실 — 기능 설명서

## 개요

**체스 교실**은 React 기반의 웹 체스 학습 앱입니다. 1인용 AI 대전과 2인용 대면 대결을 지원하며, 초보자가 체스를 배울 수 있도록 힌트·AI 해설·기물 설명 기능을 제공합니다.

- 배포 URL: https://kimgun1981.github.io/chess-learning-app/
- GitHub: https://github.com/kimgun1981/chess-learning-app

---

## 화면 구성

### 1. 메인 메뉴 (ModeSelect)
- 앱 진입 시 가장 먼저 표시
- **1인용 (AI 연습)** / **2인용** 모드 선택

### 2. 난이도 선택 (DifficultySelect) — 1인용 전용
- 초급 🌱 / 중급 ⚡ / 심화 🔥 3단계

### 3. 시간 선택 (TimeSelect) — 2인용 전용
- 무제한 / 1분 / 3분 / 5분 / 10분 / 30분

### 4. 게임 화면
- 모드에 따라 AI 모드 또는 2인용 모드 UI 표시

## 핵심 기능

### 체스판
- 8×8 그리드, 유니코드 기물 렌더링
- **흰 말**: ♙♘♗♖♕♔ (hollow, text-white + 다크 아웃라인)
- **검정 말**: ♟♞♝♜♛♚ (solid, text-gray-900 + 화이트 글로우)
- 클릭한 기물의 합법적 이동 칸 하이라이트 (파란 점)
- 체크 상태 킹 칸 빨간 하이라이트
- 힌트 칸 노란 하이라이트
- `grid-rows-8` + `h-full` 로 말 유무와 무관하게 칸 크기 고정

### AI 엔진
| 난이도 | 알고리즘 |
|--------|---------|
| 초급 | 완전 랜덤 |
| 중급 | 그리디 (체크메이트 > 체크 > 기물 잡기 > 중앙 점령 우선) |
| 심화 | 미니맥스 depth=3 + 알파베타 가지치기 |

### 타이머 (2인용)
- 선택한 시간을 각 플레이어에게 배분
- 자기 차례일 때만 카운트다운
- 30초 이하 + 자기 차례 → 빨간 색 + pulse 애니메이션
- 시간 초과 시 상대방 승리 선언

### 힌트 (1인용)
- 중급 AI와 동일한 그리디 알고리즘으로 현재 최선 수 계산
- 출발 칸·도착 칸 노란 하이라이트로 표시

### 무르기 (Undo)
- 1인용: 본인 수 + AI 수 한 쌍(2수) 동시 취소
- 2인용: 마지막 1수 취소

### AI 해설 (1인용)
- AI 착수 후 한국어로 수의 의미 자동 설명
- 체크메이트 / 체크 / 특수 이동(캐슬링, 앙파상) / 기물 잡기 / 중앙 점령 구분

### 학습 팁
- 게임 진행 수에 따라 단계별 전략 팁 표시
  - 0~5수: 오프닝 (중앙 폰 전진)
  - 6~13수: 기물 전개 및 캐슬링
  - 14~29수: 미들게임 (룩 활용)
  - 30수~: 엔드게임 (킹 활용, 폰 프로모션)

### 기물 설명
- 1인용 모드에서 기물 클릭 시 이동 규칙 한국어 설명 표시

### 기보 (Move History)
- SAN 표기법으로 전체 착수 기록 표시
- 백/흑 2열 그리드 형태

## 반응형 레이아웃

| 화면 크기 | 레이아웃 |
|-----------|---------|
| 모바일 (< 768px) | 세로 단일 컬럼 |
| 태블릿/iPad (≥ 768px) | 보드 + 사이드 패널 2단 |
| 데스크탑 (≥ 1024px) | 보드 + 넓은 사이드 패널 |

- `html, body, #root { height: 100% }` — 전체 화면 활용

## 2인용 특수 레이아웃

```
┌─────────────────────────────────┐
│  흑 (Black) 카드  [180° 회전]    │  ← 북쪽 플레이어가 읽을 수 있도록
├─────────────────────────────────┤
│                                 │
│          체 스 판                │  ← 고정 방향 (rank 8 위, rank 1 아래)
│   흑 기물(rank 7-8) 위           │
│   백 기물(rank 1-2) 아래         │
│                                 │
├─────────────────────────────────┤
│  백 (White) 카드                 │  ← 남쪽 플레이어가 읽을 수 있도록
├─────────────────────────────────┤
│  [무르기]  [새 게임]              │
└─────────────────────────────────┘
```

- 보드 방향은 차례가 바뀌어도 변경되지 않음 (고정)
- 흑 카드만 `transform: rotate(180deg)` 적용
